Claude Code とは？

Claude Code は、Anthropic が提供するコマンドライン型のAIアシスタントです。普段使っている Claude（チャット画面）とは違い、ターミナルの中で直接動きます。

最大の特徴は、会話するだけでファイルを作成・編集したり、コマンドを実行できること。「こういうサイト作って」と日本語で頼むだけで、HTML・CSS・JavaScript を書いて本当に動く Web サイトを作ってくれます。

今日のゴール

このワークショップが終わる頃には、あなた自身のポートフォリオサイトが手元で完成しています。名前、自己紹介、スキル、実績などを載せた、ブラウザで見られる本物の Web サイトです。

💡 ポイント

プログラミングの知識は一切不要です。大切なのは「自分のサイトにどんな情報を載せたいか」を考えること。今日は「日本語で指示を出す力」がそのまま Web 制作スキルになります。

Milestone 0

事前準備：環境構築

⏱ 約 15分

まず Claude Code を動かすために必要なソフトウェアをインストールします。ターミナルにコマンドを貼り付けて Enter を押すだけです。

0-1. ターミナルを開く

Mac には「ターミナル」というアプリが最初から入っています。

方法A：⌘ + スペース で Spotlight を開き、「ターミナル」と入力して Enter

方法B：Finder → アプリケーション → ユーティリティ → ターミナル.app

💡 ヒント

黒い（または白い）画面が表示されれば OK です。ここに指示を「貼り付けて Enter」するだけなので安心してください。

0-2. Node.js をインストールする

Claude Code を動かすには Node.js（バージョン18以上）が必要です。まず確認しましょう。

Terminal
node --version

v18.x.x や v20.x.x のように表示されたらインストール済みです。「command not found」と出たら、以下でインストールします。

Terminal — Homebrew 経由
# Homebrew が入っていない場合、まず Homebrew をインストール
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Node.js をインストール
brew install node

または Node.js 公式サイト（https://nodejs.org）から「LTS」版のインストーラーをダウンロードしてもOKです。

0-3. Claude Code をインストールする

Terminal
npm install -g @anthropic-ai/claude-code

インストールが完了したら確認しましょう。

Terminal
claude --version

0-4. ログインする

Terminal
claude

ブラウザが自動で開くので、Anthropic アカウントでログインしてください。完了するとターミナルに戻って Claude Code が使えるようになります。

✅ Milestone 0 完了チェック

・ターミナルが開いている
・node --version でバージョンが表示される
・claude --version でバージョンが表示される

Milestone 1

はじめの一歩：Claude Code と会話する

⏱ 約 10分

環境構築おつかれさまでした！ここからが楽しいパートです。まずは Claude Code に慣れましょう。

1-1. 作業フォルダを作って移動する

Terminal
mkdir ~/my-portfolio
cd ~/my-portfolio

1-2. Claude Code を起動する

Terminal
claude

入力待ちの状態になったら、日本語で話しかけてみましょう。

1-3. ウォーミングアップ

まずは簡単なお願いをして感覚を掴みます。

Claude Code に入力
> このフォルダの中身を教えて

まだ空のはずです。次に、簡単なファイルを作ってもらいましょう。

Claude Code に入力
> 「hello.html」を作って。中に「こんにちは、私のサイトです！」と大きく表示されるHTMLを書いて

ファイル作成の許可を求めてくるので、y を押してください。次に、作ったファイルをブラウザで確認してみましょう。

Claude Code に入力
> 今作ったファイルをブラウザで開いて

ブラウザに「こんにちは、私のサイトです！」と表示されたら成功です！これがあなたの最初の Web ページです。

ℹ️ Chat の Claude との違い

Claude Code は「ターミナルの中で動く Claude」です。ファイルの作成、コードの実行、ブラウザで開くなどパソコン上の操作を会話しながら実行できます。チャットの Claude が「コードを見せる」だけなのに対し、Claude Code は「実際にファイルを作って動かす」ことができます。

1-4. 覚えておくと便利なコマンド

コマンド説明
/helpヘルプを表示
/clear会話履歴をクリア
Ctrl + C現在の処理を中止
/exitClaude Code を終了
Esc × 2入力途中のテキストを破棄
✅ Milestone 1 完了チェック

・Claude Code で会話ができた
・HTML ファイルが作られた
・ブラウザで自分のページが表示された

Milestone 2

本番：ポートフォリオサイトを作る

⏱ 約 40分

いよいよ本番です！ Milestone 1 で作った hello.html は練習用だったので、ここからは本格的な自分のポートフォリオサイトを作っていきます。

2-1. 自分の情報を整理する（2分）

Claude Code に渡す前に、載せたい内容をざっくり決めておきましょう。メモ帳やノートに書き出してみてください。

項目例
名前田中 花子
肩書き / 一言SNSマーケター / 「届く言葉を、届く場所に」
自己紹介（2〜3行）SNSマーケティング歴3年。美容・食品ブランドを中心に…
スキル・得意分野Instagram運用、TikTok広告、コンテンツ企画
実績（2〜3個）化粧品ブランドA：フォロワー 1万→5万 に成長 など
連絡先メールアドレス、X アカウントなど
💡 ヒント

全部埋まらなくても大丈夫！最低限「名前」「肩書き」「自己紹介」があれば形になります。あとから追加・修正できるので、まずは手を動かしましょう。

2-2. Claude Code にサイトを作ってもらう

以下の指示をコピーし、★の部分を自分の情報に書き換えてから Claude Code に貼り付けてください。

Claude Code に入力（★を自分の情報に書き換える）
hello.html は削除して、以下の情報をもとにポートフォリオサイトを作って。
ファイル名は index.html で、1ファイルで完結するようにして（CSS/JSも中に書く）。

【私の情報】
- 名前: ★ 自分の名前
- 肩書き: ★ 自分の肩書き / キャッチコピー
- 自己紹介: ★ 2〜3行の自己紹介
- スキル: ★ スキルをカンマ区切りで
- 実績:
  - ★ 実績1
  - ★ 実績2
  - ★ 実績3
- 連絡先: ★ メールやSNSのURL

【デザインの要望】
- モダンでおしゃれなデザイン
- ダークテーマ
- スクロールアニメーション付き
- スマホでもきれいに見える（レスポンシブ対応）
- セクション: ヒーロー（名前＋肩書き）、自己紹介、スキル、実績、連絡先

Enter を押すと Claude Code が作業を開始します。ファイルの作成・削除の許可を求められたら y を押してください。1〜2分ほどで完成します。

2-3. ブラウザで確認する

Claude Code に入力
> ブラウザで開いて

自分の名前やプロフィールが載ったポートフォリオサイトがブラウザに表示されるはずです！

2-4. 気になるところを直す

最初の出力を見て「もっとこうしたい」と思ったら、そのまま日本語で伝えましょう。これが Claude Code の真骨頂です。

修正の指示例
> ヒーローセクションの背景をもっと暗くして、名前をもっと大きくして
修正の指示例
> スキルのセクションをカード型のレイアウトに変えて
修正の指示例
> 実績の部分に数字を大きく表示して、インパクトを出して
修正の指示例
> 全体の配色をもう少し温かみのある感じにして

修正のたびにブラウザをリロード（⌘ + R）すると、変更が反映されます。納得いくまで何度でも調整できます。

✅ Milestone 2 完了チェック

・自分の情報が載ったポートフォリオサイトがブラウザに表示されている
・最低1回は修正の指示を出して、反映された
・「日本語で言うだけでサイトが変わる」感覚を掴んだ

Milestone 3

応用：デザインと中身をカスタマイズ

⏱ 約 20分

基本のサイトが完成したら、ここからは自由時間です。自分だけのサイトに仕上げていきましょう。

カスタマイズアイデア集

Idea A

好きなデザインテイストに変更する

参考にしたいサイトの雰囲気を伝えてみましょう。

指示例
> 全体をAppleのサイトのようなミニマルで洗練された雰囲気にして。余白を多めに取って、フォントもスッキリさせて
Idea B

セクションを追加する

ブログ一覧、趣味、タイムライン（経歴）など新しいセクションを足してみましょう。

指示例
> 「経歴」セクションを追加して。タイムライン形式で表示して。
内容:
- 2020年: ○○大学卒業
- 2020年: △△株式会社 入社
- 2022年: マーケティング部に異動
- 2024年: チームリーダーに昇格
Idea C

インタラクションを追加する

ホバーエフェクトやスクロールアニメーションを追加できます。

指示例
> スキルのカードにホバーすると少し浮き上がるアニメーションを付けて。スクロールしたらセクションがふわっと表示されるようにして
Idea D

プロフィール写真を追加する

画像ファイルをフォルダに入れて、サイトに表示してもらいましょう。

指示例
> このフォルダに photo.jpg という画像があります。ヒーローセクションの右側に丸くトリミングして表示して
💡 うまくいかないときのコツ

指示はできるだけ具体的に。「おしゃれにして」よりも「背景を黒にして、アクセントカラーをオレンジにして、フォントはゴシック系で」の方が思い通りの結果になります。

・自分なりのカスタマイズを最低1つ加えた
・「日本語の指示＝Webデザイン」という感覚が掴めた
・自分だけのポートフォリオサイトが完成した！

困ったときは

「command not found: node」と出る

Node.js がインストールされていません。ターミナルを一度閉じて開き直し、再度 node --version を試してください。ダメなら Node.js 公式サイトからインストーラーで再インストールしましょう。

「command not found: claude」と出る

npm install -g @anthropic-ai/claude-code を再度実行してください。npm 自体が使えない場合は Node.js のインストールから確認しましょう。

ブラウザで開いてもページが真っ白

HTML にエラーがある可能性があります。Claude Code に「ブラウザで開いたら真っ白だった。HTMLのエラーを確認して修正して」と伝えましょう。自動で修正してくれます。

Claude Code がフリーズしたように見える

大きな処理中は時間がかかることがあります。1〜2分待ってみてください。それでもダメなら Ctrl + C で中止し、もう一度指示を出しましょう。

デザインが思い通りにならない

「もっと具体的に」指示を出してみてください。色コードやサイズを指定すると正確になります。参考にしたいサイトのURLを貼って「この雰囲気を参考にして」と伝えるのも有効です。

何を指示していいかわからなくなった

「今のサイトの改善点を3つ提案して」と聞いてみましょう。Claude Code がデザインや構成の改善案を出してくれます。

完成したサイトを公開するには

作ったポートフォリオサイトはあなたの Mac 上にある index.html ファイルです。インターネットに公開する方法はいくつかあります。

おすすめ

GitHub Pages（無料）

GitHub アカウントを作ってファイルをアップロードするだけで公開できます。Claude Code に「このサイトを GitHub Pages で公開する手順を教えて」と聞けば、手順を案内してくれます。

かんたん

Netlify Drop（無料）

netlify.com/drop にアクセスし、フォルダをドラッグ＆ドロップするだけで公開できます。最も簡単な方法です。
