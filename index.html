そもそも Claude（クロード）って何？

Claude は、Anthropic（アンソロピック）というアメリカの会社が作ったAIアシスタントです。ChatGPT と同じジャンルのサービスで、日本語で話しかけるといろんなことを手伝ってくれます。

ℹ️ ChatGPT と何が違うの？

車に例えると、トヨタとホンダの違いのようなもの。どちらも優秀な AI ですが、Claude は「嘘をつきにくい」「長い文章に強い」「安全性を重視している」という特徴があります。そして今回使う「Cowork」という仕組みがとても強力です。

Claude には「3つの顔」がある

Claude にはモードが3つあります。これを理解しておくと、今日やることがスッキリわかります。

1

Claude チャット ＝ 優秀な「相談相手」

ブラウザやアプリで使える会話モード。質問すると答えてくれる。ただし最後の作業は自分でやる必要がある。「こういうコードを書いてください」と教えてくれるけど、実際にファイルを作るのは自分。ChatGPT と同じ立ち位置。

2

Claude Code ＝ 頭脳に「手足」を与えたもの

ターミナル（黒い画面）で動く、最も高機能なモード。ファイル操作、ツール連携、複数のAIを同時に指揮する——なんでもできる。ただしターミナルを使うので、エンジニア向け。名前に「Code」とあるがコーディング以外にも使える万能ツール。

3

Cowork ＝ Claude Code の「簡易版」

Claude デスクトップアプリの中にある作業モード。Claude Code と同じ仕組みで動いているが、ターミナル不要で、チャット画面から指示するだけで使える。Claude Code ほど高機能ではないが、ファイル作成・編集・整理など日常業務には十分。非エンジニア向けに作られた入口。

💡 わかりやすく言うと

Claude の「AI モデル」は頭脳。チャットはその頭脳と「会話」する窓口。Claude Code はその頭脳に「手足」を与えたもの。そして Cowork はClaude Code の基本機能をターミナルなしで使える簡易版。
Cowork でできることは Claude Code で全部できる。逆に Claude Code の方が圧倒的に高機能。でも、ターミナルが怖い人にとっては Cowork が最高の入口になる——それが今日使うツールです。

Milestone 0

事前準備：Cowork を使えるようにする

⏱ 約 10分

Cowork を使うための準備は3ステップだけ。ターミナルもプログラミングも不要です。

0-1. Claude デスクトップアプリをインストールする

1

claude.ai/download にアクセスする

2

お使いのOS（Mac / Windows）に合ったアプリをダウンロード

3

ダウンロードしたファイルを開いてインストール（Mac の場合はアプリケーションフォルダにドラッグ）

ℹ️ Windows の場合

Windows で Cowork を使うには「開発者モード」をオンにする必要があります。設定 → システム → 開発者向け → 「開発者モード」をオンにしてください。対応しているのは x64（64ビット版）のみです。

0-2. アカウントを用意する

1

claude.ai にアクセスし、アカウントを作成する（メールアドレスまたは Google アカウントで登録）

2

Pro プラン（月額 $20）に加入する — Cowork は無料プランでは使えません

3

Claude デスクトップアプリを開いて、作成したアカウントでログインする

ℹ️ Pro プランについて

Cowork を使うには Pro プラン以上が必要です。月額 $20（約 3,000 円）で、Cowork・Research・最新モデルなどすべての機能が使えます。まずは1ヶ月試してみて、合わなければ解約できます。

0-3. 作業用フォルダを準備する

Cowork はフォルダ単位で仕事をします。指定したフォルダの中身だけを読み書きするので、フォルダの作り方がそのまま仕事の整理になります。

まず、Cowork の「ホーム」になるフォルダを1つだけ手動で作ります。

1

Finder（Mac）またはエクスプローラー（Win）を開き、自分のユーザーフォルダに移動する
Mac: Finder → 移動 → ホーム ／ Win: C:\Users\自分の名前

2

そこに 「Claude-Workspace」 フォルダを1つだけ新規作成する

💡 中のフォルダは Cowork に作ってもらいます

ここで手動で作るのはこの1つだけ。中身のフォルダ構成は、この後 Cowork に指示して作ってもらいます。それが最初の「AI に働いてもらう」体験になります。

0-4. Cowork モードに切り替える

1

アプリ上部にある「Chat」「Cowork」のタブを見つける

2

「Cowork」タブをクリックして切り替える

3

作業用フォルダを選択する画面が出るので、先ほど作った「Claude-Workspace」フォルダを選択する

ℹ️ Chat と Cowork の違い

Chat は質問に答えてくれる「相談相手」。Cowork はファイルを実際に作成・編集してくれる「作業パートナー」です。今日使うのは Cowork の方です。

0-5. 最初の指示：フォルダ構成を作ってもらう

さっそく Cowork に最初の仕事を頼みましょう。以下をコピーして Cowork に貼り付けてください。

Cowork に入力
このフォルダの中に、以下の3つのサブフォルダを作って。

- context   （私の自己紹介や好みなど、あなたに知っておいてほしい情報を置く場所）
- projects  （進行中のプロジェクトを入れる場所）
- outputs   （完成した成果物を保存する場所）

Cowork がフォルダを作成します。許可を求められたら許可してください。

できたら、Finder やエクスプローラーで Claude-Workspace を開いてみてください。3つのフォルダが実際にできているはずです。これが「AI に働いてもらう」ということです。

完成イメージ
Claude-Workspace/
  ├── context/     ← 自分の情報を入れておく場所
  ├── projects/    ← 進行中のプロジェクト（今日のポートフォリオもここ）
  └── outputs/     ← 完成した成果物の保存先
💡 なぜこの3つなのか

context/ がこの構成の肝です。ここに自己紹介やプロフィール情報を置いておくと、Cowork は仕事を頼まれるたびにこのフォルダを読みに行けるので、「私のことを知っている部下」として働いてくれます。毎回「私は○○で、仕事は○○で…」と説明し直す必要がなくなります。

projects/ にはプロジェクトごとのサブフォルダを作っていきます。今日のポートフォリオもここに入ります。プロジェクトが増えても構造を変える必要がありません。

outputs/ は「これは完成品」とわかるものだけ置く場所。作業中のファイルと完成品が混ざらないので管理が楽になります。

✅ Milestone 0 完了チェック

・Claude デスクトップアプリが開いている
・自分のアカウントでログインできている
・「Cowork」タブに切り替わっている
・Cowork にフォルダ作成を指示して、実際に3つのフォルダができた
→ おめでとうございます！これが初めての「AI に働いてもらう」体験です

Milestone 1

はじめの一歩：Cowork でサイトを作る準備

⏱ 約 10分

Milestone 0 で Cowork に初めて仕事を頼みました。ここからはポートフォリオサイトの制作に入っていきます。

1-1. projects フォルダ内にポートフォリオ用フォルダを作る

今日のポートフォリオサイトは projects/ の中に作ります。Cowork に頼みましょう。

Cowork に入力
projects フォルダの中に「portfolio」というサブフォルダを作って

できたら、フォルダを切り替えます。

1

Cowork 画面の上部にあるフォルダ名をクリック

2

「Claude-Workspace」→「projects」→「portfolio」フォルダを選択し直す

1-2. ウォーミングアップ

Cowork の入力欄に、日本語で話しかけてみましょう。まずは簡単なお願いから。

Cowork に入力
> このフォルダの中身を教えて

まだ空のフォルダなので、「ファイルはありません」と教えてくれるはずです。次に、簡単なファイルを作ってもらいましょう。

Cowork に入力
> 「hello.html」を作って。中に「こんにちは、私のサイトです！」と大きく表示されるHTMLを書いて

Cowork がファイルを作成します。作成の許可を求めてくることがあるので、許可してください。

1-3. 作ったファイルをブラウザで見る

Cowork に頼んで、作ったファイルをブラウザで開きましょう。

Cowork に入力
> 今作ったファイルをブラウザで開いて

ブラウザに「こんにちは、私のサイトです！」と表示されたら成功です！これがあなたの最初の Web ページです。

もし Cowork がブラウザで開けなかった場合は、「Claude-Workspace/projects/portfolio」フォルダを開いて、hello.html をダブルクリックしても OK です。

ℹ️ いま体験したのが「相談相手」と「部下」の違い

Chat の Claude に同じことを頼むと「こういうコードを書いてください」とコードを見せてくれるだけ。コピペして保存するのは自分。Cowork は実際にファイルを作って、フォルダに保存してくれる。頼んだら完成品が返ってくる——これが「質問する AI」と「働いてくれる AI」の違いです。

✅ Milestone 1 完了チェック

・Cowork で会話ができた
・HTML ファイルが作られた
・ブラウザで自分のページが表示された

Milestone 2

本番：ポートフォリオサイトを作る

⏱ 約 40分

いよいよ本番です！ Milestone 1 で作った hello.html は練習用だったので、ここからは本格的な自分のポートフォリオサイトを作っていきます。

2-1. 自分の情報を整理する（2分）

Cowork に渡す前に、載せたい内容をざっくり決めておきましょう。メモ帳やノートに書き出してみてください。

項目例
名前田中 花子
肩書き / 一言SNSマーケター / 「届く言葉を、届く場所に」
自己紹介（2〜3行）SNSマーケティング歴3年。美容・食品ブランドを中心に…
スキル・得意分野Instagram運用、TikTok広告、コンテンツ企画
実績（2〜3個）化粧品ブランドA：フォロワー 1万→5万 に成長 など
連絡先メールアドレス、X アカウントなど
💡 ヒント

全部埋まらなくても大丈夫！最低限「名前」「肩書き」「自己紹介」があれば形になります。あとから追加・修正できるので、まずは手を動かしましょう。

2-2. Cowork にサイトを作ってもらう

以下の指示をコピーし、★の部分を自分の情報に書き換えてから Cowork に貼り付けてください。

Cowork に入力（★を自分の情報に書き換える）
hello.html は削除して、以下の情報をもとにポートフォリオサイトを作って。
ファイル名は index.html で、1ファイルで完結するようにして（CSS/JSも中に書く）。

【私の情報】
- 名前: ★ 自分の名前
- 肩書き: ★ 自分の肩書き / キャッチコピー
- 自己紹介: ★ 2〜3行の自己紹介
- スキル: ★ スキルをカンマ区切りで
- 実績:
  - ★ 実績1
  - ★ 実績2
  - ★ 実績3
- 連絡先: ★ メールやSNSのURL

【デザインの要望】
- モダンでおしゃれなデザイン
- ダークテーマ
- スクロールアニメーション付き
- スマホでもきれいに見える（レスポンシブ対応）
- セクション: ヒーロー（名前＋肩書き）、自己紹介、スキル、実績、連絡先

送信すると Cowork が作業を開始します。ファイルの作成・削除の許可を求められたら許可してください。1〜2分ほどで完成します。

2-3. ブラウザで確認する

Cowork に入力
> ブラウザで開いて

自分の名前やプロフィールが載ったポートフォリオサイトがブラウザに表示されるはずです！

ブラウザで開けない場合は、「Claude-Workspace/projects/portfolio」フォルダにある index.html をダブルクリックしてください。

2-4. 気になるところを直す

最初の出力を見て「もっとこうしたい」と思ったら、そのまま日本語で伝えましょう。これが Cowork の真骨頂です。

修正の指示例
> ヒーローセクションの背景をもっと暗くして、名前をもっと大きくして
修正の指示例
> スキルのセクションをカード型のレイアウトに変えて
修正の指示例
> 実績の部分に数字を大きく表示して、インパクトを出して
修正の指示例
> 全体の配色をもう少し温かみのある感じにして

修正のたびにブラウザをリロード（Mac: ⌘ + R / Win: F5）すると、変更が反映されます。納得いくまで何度でも調整できます。

✅ Milestone 2 完了チェック

・自分の情報が載ったポートフォリオサイトがブラウザに表示されている
・最低1回は修正の指示を出して、反映された
・「日本語で言うだけでサイトが変わる」感覚を掴んだ

Milestone 3

応用：デザインと中身をカスタマイズ

⏱ 約 20分

基本のサイトが完成したら、ここからは自由時間です。自分だけのサイトに仕上げていきましょう。

カスタマイズアイデア集

Idea A

好きなデザインテイストに変更する

参考にしたいサイトの雰囲気を伝えてみましょう。

指示例

セクションを追加する

ブログ一覧、趣味、タイムライン（経歴）など新しいセクションを足してみましょう。

指示例
> 「経歴」セクションを追加して。タイムライン形式で表示して。
内容:
- 2020年: ○○大学卒業
- 2020年: △△株式会社 入社
- 2022年: マーケティング部に異動
- 2024年: チームリーダーに昇格
Idea C

インタラクションを追加する

ホバーエフェクトやスクロールアニメーションを追加できます。

指示例

Idea D

プロフィール写真を追加する

画像ファイルをフォルダに入れて、サイトに表示してもらいましょう。

指示例

指示はできるだけ具体的に。「おしゃれにして」よりも「背景を黒にして、アクセントカラーをオレンジにして、フォントはゴシック系で」の方が思い通りの結果になります。

・自分なりのカスタマイズを最低1つ加えた

困ったときは

Cowork タブが表示されない

Claude デスクトップアプリが最新版であることを確認してください。アプリを再起動するか、claude.ai/download から最新版をダウンロードし直してみましょう。また、Cowork は Pro プラン以上でないと使えません。

「フォルダを選択してください」と出るが選べない

ユーザーフォルダ（自分の名前のフォルダ）に「Claude-Workspace」フォルダを手動で作っておく必要があります。中のサブフォルダは Cowork に作ってもらうので、まずは空の「Claude-Workspace」だけ作ってください。

ファイルが作られたはずだがフォルダに見つからない

Cowork が作業するフォルダと「Claude-Workspace/projects/portfolio」が正しくリンクされているか確認しましょう。Cowork 画面の上部にフォルダのパスが表示されています。

ブラウザで開いてもページが真っ白

HTML にエラーがある可能性があります。Cowork に「ブラウザで開いたら真っ白だった。HTMLのエラーを確認して修正して」と伝えましょう。自動で修正してくれます。

Cowork の応答が遅い・止まったように見える

大きな処理中は時間がかかることがあります。1〜2分待ってみてください。それでもダメなら新しいセッションを開始（左上の「＋」ボタン）して、同じフォルダを選び直しましょう。

デザインが思い通りにならない

「もっと具体的に」指示を出してみてください。色コードやサイズを指定すると正確になります。参考にしたいサイトのURLを貼って「この雰囲気を参考にして」と伝えるのも有効です。

何を指示していいかわからなくなった

「今のサイトの改善点を3つ提案して」と聞いてみましょう。Cowork がデザインや構成の改善案を出してくれます。

Windows で Cowork が動かない

「開発者モード」がオンになっているか確認してください（設定 → システム → 開発者向け）。また、Windows ARM64 版（Qualcomm チップ搭載PC）には未対応です。x64（64ビット版）のみ対応しています。

完成したサイトを公開するには

作ったポートフォリオサイトはあなたのパソコン上にある index.html ファイルです。インターネットに公開する方法はいくつかあります。

いちばん簡単

Netlify Drop（無料）

netlify.com/drop にアクセスし、「portfolio」フォルダをドラッグ＆ドロップするだけで公開できます。アカウント不要で最も簡単な方法です。

おすすめ

GitHub Pages（無料）

GitHub アカウントを作ってファイルをアップロードするだけで公開できます。Cowork に「このサイトを GitHub Pages で公開する手順を教えて」と聞けば、手順を案内してくれます。

別のサイトを作る：クライアント向けLPやチーム紹介ページなど業務で使えるサイトに挑戦
